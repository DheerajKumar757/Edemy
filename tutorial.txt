For `rafce` code snippet use extension `ES7 React/Redux/GraphQL/React-Native snippets`

To run the frontend :
go to `client` folder and run `npm run dev`

packages :
`react-router-dom` to use routing on in react project
`react-youtube` to add youtube videos on the website
`uniqid` for unique hexadecimal ids of courses
`quill` to generate rich texts for course packages
`humanize-duration` to convert videos duration in hours min and seconds
`rc-progress` to design circular progess or progess bar
`react-simple-star-rating` to display star rating
